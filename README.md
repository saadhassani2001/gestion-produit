# Gestion de Stock de Pharmacie (C)

## 📌 Description
Ce projet est une application en langage C permettant de gérer le stock d’une pharmacie.  
Il permet d’ajouter, modifier, supprimer et rechercher des produits.

## ⚙️ Fonctionnalités
- Ajouter un produit
- Supprimer un produit
- Modifier un produit
- Afficher tout le stock
- Rechercher un produit par code
- Vérifier les alertes de faible quantité

## 🧠 Technologies utilisées
- Langage C
- Structures (struct)
- Tableaux
- Fonctions

## 🚨 Système d’alerte
Le programme affiche une alerte si la quantité d’un produit est inférieure à 5.
